3. python 的环境
    编译型：
        一次性将所有的程序 编译成二进制文件。
        缺点：开发效率低，不能跨平台。
        优点：执行速度快
        如：C, C++ 等等
        
    解释型：
        当程序执行时，一行一行的解释。
        优点：开发效率高，可以跨平台。
        缺点：运行速度慢。
        如：python, php, 等等
